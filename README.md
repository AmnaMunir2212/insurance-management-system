# Insurance Management System

A **Java Desktop Application** developed using **Java Swing** to manage insurance-related operations.
The system provides a simple graphical user interface (GUI) for managing customers, agents, policies, and insurance records.

## ✨ Features

* Customer Management
* Agent Management
* Insurance Policy Management
* Search Functionality
* User-friendly Java Swing GUI
* Object-Oriented Programming (OOP) concepts

## 🛠️ Technologies Used

* Java
* Java Swing
* NetBeans IDE
